# moj_asystent_zdrowia

#opis 
Aplikacja mobilna została zaprojektowana w Flutterze i służy do wspierania użytkownika w monitorowaniu zdrowia oraz codziennych nawyków. Umożliwia analizę podstawowych danych zdrowotnych oraz ich wykorzystanie do prostych obliczeń.

#główne funkcjonalności
1. rejestr leków
aplikacja umożliwia:
-dodawać przyjmowane leki, ich dawkę i nazwę,
-potwierdzenie wzięcia leku

2. monitorowanie nawodnienia
aplikacja umożliwia:
-zapisywanie ilości wypitej wody,
-śledzenie postępów w ciągu dnia

3. zarządzanie wizytami lekarskimi
aplikacja umożliwia:
-dodawanie wizyt lekarskich, 
-edycję zapisanych wizyt, 
-usuwanie wizyt,
-przechowywanie informacji takich jak lekarz, miejsce wizyty, data, notatki

4. kalkulację bmi i zapotrzebowania na wodę
-na podstawie podanego przez użytkownika wagi, wzrostu i wieku

#przechowywanie danych
Dane są zapisywane lokalnie w urządzeniu użytkownika przy użyciu SharedPreferences, co zapewnia szybki dostęp do danych, brak konieczności logowania, działanie offline

#cel aplikacji
Celem aplikacji jest:
-wspieranie codziennej kontroli zdrowia,
-ułatwienie monitorowania nawyków
-dostarczanie prostych, automatycznych obliczeń zdrowotnych,
-posiadanie podstawowych danych medycznych w jednym miejscu
